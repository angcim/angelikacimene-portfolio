Brain Frame

This was a final project for my processing course. The object was to code and generate an interactive design that gives users a knowledge and information about a topic. It should provide natural designs and aesthetics to create a pleasing interactive design.

Brain Frame is an interactive design that navigates the users through different parts of the brain. Brain Frame educates users about the different functions and operations of the brain through ambient music and psychedellic visuals.

How it works: The user hovers on each section of the brain (which are the boxes) to receive a visual and sound. When the user clicks on that section, it provides information about the importance, function, and in some, how that area responds to music. To better enhance the user's experience, they can press a numbers from 1-5 and it will provide more visuals.

Programs Used: Processing
